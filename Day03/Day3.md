    do this

2. Nested if/else

if condition:
    if another condition:
        do this
    elif condition2:
        do this
    else:
        do this

    do this


4. Multiple if statements

if condition1:
    do A
if condition2:
    do B
if condition3:
    do C

not E    --negation
